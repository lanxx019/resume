# Resume
My Resume. Based on [harnon-cv](https://www.ctan.org/pkg/harnon-cv?lang=en).

## Requirements:
- [MacTex](http://www.tug.org/mactex/index.html) via `brew`. Note you might need to open a new terminal.
- Need to install `fontawesome` in through `tlmgr`:
  ```
  sudo tlmgr install fontawesome
  ```

## Build
To build the PDF, simply run `make`.
